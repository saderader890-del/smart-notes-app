# 📝 Smart Notes App

Полнофункциональное мобильное приложение для создания заметок с поддержкой ИИ-генерации тестов и умного чата.

## ✨ Основные возможности

- 📝 **Заметки** — создание, редактирование, поиск, теги
- 🤖 **ИИ-ассистент** — чат с контекстом ваших заметок, резюме
- 📋 **Тесты** — создание вручную или автоматическая генерация из заметок
- 💾 **Синхронизация** — сохранение в LocalStorage

## 🚀 Быстрый старт

### 1. Получить Groq API ключ

1. Перейдите на https://console.groq.com
2. Зарегистрируйтесь / войдите
3. Перейдите в **API Keys**
4. Нажмите **Create API Key**
5. Скопируйте ключ (выглядит так: `gsk_xxxxxxxxxxxxx`)

### 2. Клонировать репозиторий

```bash
git clone https://github.com/saderader890-del/smart-notes-app.git
cd smart-notes-app
```

### 3. Установить зависимости

```bash
npm install
```

### 4. Создать .env файл

Создайте файл `.env` в корне проекта:

```env
REACT_APP_GROQ_API_KEY=gsk_ваш_api_ключ_здесь
```

**Важно:** Замените `gsk_ваш_api_ключ_здесь` на ваш реальный ключ из Groq!

### 5. Запустить приложение

```bash
npm start
```

Приложение откроется на `http://localhost:3000`

## 📱 Использование

### Заметки
- **Новая заметка** — нажмите "Новая"
- **Поиск** — используйте строку поиска (ищет по названию, содержимому, тегам)
- **Теги** — добавляйте теги для организации заметок

### ИИ-ассистент
- **Чат** — задавайте вопросы, ИИ имеет доступ к вашим заметкам
- **Резюме** — откройте заметку и нажмите "ИИ-резюме и вопросы"

### Тесты
- **Создать вручную** — нажмите "Создать" на вкладке Тесты
- **Из заметки** — откройте заметку и нажмите "Создать тест с ИИ"
- **Пройти тест** — нажмите на тест и отвечайте на вопросы
- **Результаты** — сохраняются автоматически

## 🎨 Технологии

- **React 18** — интерфейс
- **Groq API** — ИИ для генерации тестов и чата
- **LocalStorage** — сохранение данных

## 🔒 Безопасность

- **Никогда не коммитьте файл `.env`** — он в `.gitignore`
- **Не делитесь API ключом** — добавьте его в `.env` локально
- Если ключ скомпрометирован, создайте новый на console.groq.com

## 🐛 Решение проблем

### Ошибка: "API ключ не найден"
- Убедитесь, что вы создали файл `.env`
- Проверьте, что ключ скопирован правильно
- Перезагрузите приложение: `Ctrl+C` и `npm start`

### Генерация теста не работает
- Проверьте, что API ключ корректен
- Убедитесь, что интернет соединение активно
- Посмотрите консоль браузера (F12) на ошибки

### Данные теряются после перезагрузки
- Проверьте, что LocalStorage не отключён в браузере
- Попробуйте другой браузер
